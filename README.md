## Learn

- **[dora](https://github.com/dora-rs/dora)**  
  Core framework with “Getting Started,” examples (YOLO graph), docs and APIs.

- **[dora-courses](https://github.com/dora-rs/dora-courses)**  
  Structured intro course: CLI, shared memory, tracing, fundamentals.

- **[Official Guides](https://dora-rs.ai/docs/guides/)**  
  Installation, pipelines, debugging, ROS2, autonomous driving.

- **[Examples Index](https://dora-rs.ai/docs/examples/)**  
  Full index of official vision, ROS2, and tutorial examples.

- **[LLM Assistant Guide](https://dora-rs.ai/docs/guides/getting-started/llm/)**  
  Build an LLM chatbot with Whisper STT + keyboard input.

- **[dora-llm](https://github.com/dora-rs/dora-llm)**  
  Quick-start repo for LLM-based dataflows.

- **[dora-examples](https://github.com/dora-rs/dora-examples)**  
  Ready-to-run examples across languages and domains.

- **[dora-drives](https://github.com/dora-rs/dora-drives)**  
  Autonomous driving learning pipeline.
